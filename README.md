# MedAssist Avant
A healthcare app for the MedAssist hackathon.

## Tech Stack
- Frontend: Vite, React
- CI/CD: Travis CI
- Deployment: Vercel

## Setup
1. Clone: git clone https://github.com/poowa-gg/medassist-avant.git
2. Install: npm install
3. Run: npm run dev

## Pages
- Login
- Signup

- Forgot Password
- Language Preference
- Gender Preference
- Symptom Description
